Скопировать содержимое репозитория на жесткий диск.
Из папки с проектом запустить сервер: python manage runserver
Далее по ссылкам.

Добавлять обложки для книг лучше в админ-панели, но можно и в соответствующей форме на отдельной странице. Ссылки на вспомогательные страницы собраны на странице библиотеки:
http://127.0.0.1:8000/lib/
